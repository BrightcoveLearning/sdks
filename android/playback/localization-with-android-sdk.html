Overview

The Brightcove SDK for Android supports localizing the player in English and Arabic. This includes player controls, caption selection and advertising messages such as "Skip Ad" and "Ad 1 of 3".

The default layout for the Arabic language is set as right-to-left.

Working with IMA

Because the IMA SDK does not localize text automatically, you need to set it manually.

  1. First, find the current language of the device:

    Locale.getDefault().getLanguage();

  2. Then, set the language when initializing the GoogleIMAComponent:

    ImaSdkSettings settings = ImaSdkFactory.getInstance().createImaSdkSettings();
settings.setLanguage("ar");
googleIMAComponent = new GoogleIMAComponent(
    brightcoveVideoView, 
    eventEmitter, 
    settings);

Forcing left-to-right layouts

Some languages, such as Arabic, will change the layout direction, and set it as right-to-left. If this does not fit your use case, you can enforce the left-to-right layout by modifying the AndroidManifest.xml as follows:

<application
  android:name=".PlaybackTestApplication"
  android:allowBackup="true"
  android:icon="@mipmap/ic_launcher"
  android:banner="@drawable/tv_banner"
  android:label="@string/app_name"
  android:theme="@style/AppTheme"
  android:usesCleartextTraffic="true"
  android:supportsRtl="false"
  tools:replace="android:supportsRtl">
